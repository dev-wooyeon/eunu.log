# Three.js로 만드는 상호작용 경험

안녕하세요! 오늘은 Three.js를 사용하여 사용자에게 기억에 남는 경험을 선사하는 방법에 대해 알아보겠습니다.

## 왜 Three.js인가?

웹 환경에서 3D 그래픽을 구현하는 것은 매우 강력한 도구입니다. 사용자에게 몰입감을 주고, 일반적인 2D 인터페이스보다 더 깊은 인상을 남길 수 있습니다.

```javascript
import * as THREE from 'three';

const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(75, window.innerWidth / window.innerHeight, 0.1, 1000);
```

## 결론

상호작용은 단순히 기술적인 구현을 넘어 사용자와의 교감을 의미합니다.
